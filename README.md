# Indexer — красивые степени и индексы в любом чате

Устали писать `5^2` что бы написать степень над числом?  
**Indexer** автоматически превращает `5^2` в **5²** прямо на странице.

Работает в любом браузере на Chromium (Chrome, Edge, Brave, Opera)

---

## ✨ Возможности

- `5^2` → 5²  
- `x^10` → x¹⁰  

Поддерживаются:
- цифры и латинские буквы
---

## 🚀 Установка

1. Скачай папку с расширением (или склонируй репозиторий).
2. Открой браузер → `chrome://extensions/`
3. Включи **«Режим разработчика»** (правый верхний угол).
4. Нажми **«Загрузить распакованное»**.
5. Выбери папку с расширением (ту, где лежит `manifest.json`).

Готово! Расширение сразу начнёт работать.

---

## 🛠️ Как это работает

Расширение:
1. Сканирует все текстовые узлы на странице.
2. Ищет паттерны `число^число` или `буква_число`.
3. Заменяет их на HTML-теги `<sup>`

Новые сообщения в чатах обрабатываются автоматически через `MutationObserver`.
